### Task 1: Explore the Temple Entrance
Navigate to the `temple-entrance/` directory and discover what secrets it holds.

**Commands you'll need:**
- `cd` - change directory
- `ls` - list contents
- `cat` - read file contents
- `ls -la` - list all files including hidden ones

**Hint**: Not everything is visible at first glance. The ancients were masters of hiding things in plain sight.

---

### Task 2: Decode the Ancient Message
Inside the temple, you'll find encrypted messages. Use your Linux skills to decode them.

**Commands you'll need:**
- `grep` - search for patterns
- `echo` - display text
- `base64 -d` - decode base64 (if needed)

---

### Task 3: Unlock the Sealed Chamber
The sealed chamber requires specific permissions to open. You must set the correct permissions to proceed.

**Commands you'll need:**
- `chmod` - change file permissions
- `./` - execute a file
